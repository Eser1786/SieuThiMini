## Cách sử dụng các lệnh git

### git pull

lấy code trên github mới nhất về

### tổ hợp các lệnh sau dùng để đẩy code lên

1. git add .
   thêm tất cả các file mình đã chỉnh sửa có thể thêm bằng git add "Tên file"
2. git commit -m "tổng hợp cho code mình làm"

3. git push
   Đẩy code lên github
